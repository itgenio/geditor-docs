# Материалы

## Структура

Материал состоит из слайдов

Слайд состоит из объектов

Объект состоит из компонентов

## Настройки материала

- название
- автор
- рецензент?
- видимость: закрытый/организация/публичный

### Видимость

**Закрытый** материал виден только пользователю.

**Организация** - внутри организации, к которой принадлежит пользователь.

**Публичный** - всем
