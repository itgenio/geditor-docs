# Магазин

## Для чего нужен

Магазин служит местом обмена наработками, которые можно применять при создании материала.

**Пакет** - это наработка.

## Пакет

Пакет содержит:

- название
- описание
- автор
- иконка?
- версия
- набор ресурсов

Ресурс это:

- аудио, видео, изображение, текст, т.е. любой бинарный файл.
- префаб
- стиль?
- шаблон?
