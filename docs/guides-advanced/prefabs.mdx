# Префабы

**Префаб** - это "шаблон" объекта, который можно переиспользовать много раз.

**Префаб** это вид ресурса.

**Экземпляр префаба** - объект на сцене.

## Создание префаба

Префаб можно создать выбрав объект и в меню нажать "Создать префаб".

### Ограничения

Т.к. префаб - это ресурс, то он должен быть независимой единицей.

- ссылки на объекты/компоненты могут быть только из иерархии префаба. Если создается префаб и есть внешние ссылки, то они будут сброшены.

## Режим редактирования префаба

- нельзя удалять объекты из префаба

## Переопределение свойств в экземплярах

Переопределять свойства компонентов, добавлять объекты.

Свойства, которые не синхронизируются: Transform
